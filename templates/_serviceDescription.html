Hype5 Server

Built and open-source project API which serves an aggragate of data from music blogs. The API operate via compiling all requested data via a web crawling and returns response afater web crawling completes. Project runs in Docker container.

Note: Given that this project is based on PhantomJS and is crawling web address that use javascript responses will take a WHILE (avg on good connection 3-7sec #firstworldproblems). Therefore, design accordingly and use loaders and animation in places where the data is being requested.
